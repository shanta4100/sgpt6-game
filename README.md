# SSGPT6 Game

Root governance console online.

## Getting Started

This is a [Next.js](https://nextjs.org) project with Vercel Speed Insights integrated.

### Development

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Speed Insights

This project includes [Vercel Speed Insights](https://vercel.com/docs/speed-insights) for monitoring real-world performance metrics.

Speed Insights is automatically enabled once you:
1. Deploy the project to Vercel
2. Enable Speed Insights in your Vercel project dashboard
3. The `<SpeedInsights />` component is already integrated in `app/layout.tsx`

After deployment, performance data will be visible in the Speed Insights tab of your Vercel dashboard.

## Build

To create a production build:

```bash
npm run build
```
